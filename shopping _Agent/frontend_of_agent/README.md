# 🛒 Shopping Agent Frontend

This is the frontend of the Shopping Agent project.
It provides a chat-based interface where users can interact with an AI-powered virtual assistant to get product suggestions.

## 🚀 Features

💬 Chat interface with AI assistant

🎯 Product name extraction from AI responses

🖼️ Product images and suggestions displayed dynamically

📱 Responsive layout with Tailwind CSS

##  🛠️ Tech Stack

Next.js (React) – Frontend framework

TypeScript – For type safety

Tailwind CSS – Styling

Fetch API – For communication with backend

🔧 Getting Started
1. Clone the repository
git clone https://github.com/your-username/shopping-agent-frontend.git
cd shopping-agent-frontend

2. Install dependencies
npm install

3. Run the development server
npm run dev


The app will be running at http://localhost:3000
